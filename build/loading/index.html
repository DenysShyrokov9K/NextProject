加载组件，用于页面和区块的加载中状态。

何时使用 #

页面局部处于等待异步数据或正在渲染过程时，合适的加载动效会有效缓解用户的焦虑。

如何使用 #

Loading 默认使用 display='inline-block' 布局的方式包裹内部元素。

如果希望通栏包裹， 可以修改 <Loading style={{display: 'block'}} />。

代码演示 #

基本 #

简单的 Loading 状态，包裹需要显示加载态的组件。

动画尺寸 #

通过size设置Loading动画的尺寸，只对原生的动画指示符indicator有效。

自定义提示语及其位置 #

通过tip自定义加载提示语，通过tipAlign设置提示语的位置，目前支持 right / bottom (default)。

自定义动画指示符 #

通过indicator自定义动画指示符，传入动画元素。

受控关闭加载 #

通过visible受控显示或关闭加载动画。

全屏 #

通过fullScreen强制全屏显示加载态，通过safeNode指定安全节点。

无障碍支持 #

默认支持通过按下Esc键退出全屏显示加载态。
